# CQ500 颅脑CT急诊数据集详细介绍

# 一、数据集整体概述

CQ500是医学影像领域公开免费、经典权威的头部CT急诊数据集，由Qure\.ai联合印度新德里CARING影像研究中心发布。该数据集专门用于颅脑急症智能检测算法的训练、验证与临床泛化测试，是脑出血、颅脑损伤AI识别任务的主流基准数据集，可适配2D、3D医学影像分类、目标检测、图像分割等各类医学影像研究任务。

# 二、基础核心信息

- **数据规模**：数据集共包含491例头部CT扫描病例（非网传500例），总计193317张CT切片。数据采集于多家医院及独立影像中心，涵盖门诊、急诊真实临床场景，数据场景多样性强，贴合真实临床诊疗环境。

- **数据筛选标准**：统一剔除术后复查影像、7岁以下儿童病例，所有样本均为成人非对比平扫头部CT，精准适配成人颅脑急症临床研究与落地场景。

- **数据来源**：数据独立于同团队Qure25k训练数据集，分两个批次完成采集，完全规避训练集数据泄露问题，具备极高的权威性，可用于模型跨中心外部有效性验证。

- **开源协议**：遵循CC BY\-NC\-SA 4\.0开源协议，可供科研人员免费开展学术研究使用。

# 三、标注体系与类别

数据集标注工作由三位从业年限分别为8年、12年、20年的资深放射科医师完成，通过交叉标注、共识校验的方式确定金标准，标注精度与可靠性极高。数据集支持多标签分类任务，核心覆盖临床常见的9类颅脑急症异常，具体病变类型如下：

- 整体颅内出血（ICH）

- 脑实质出血（IPH）

- 脑室内出血（IVH）

- 硬膜下出血（SDH）

- 硬膜外出血（EDH）

- 蛛网膜下腔出血（SAH）

- 其他颅脑肿块、病变、骨折等急症异常

**正负样本分布**：数据集中共205例出血阳性病例，其余为正常阴性样本，样本均衡性适中，能够满足AI模型正负样本拟合训练的基本需求。
- 这些标注信息由两个文件组成：reads.csv和prediction-probabilities.csv，其中reads.csv文件保存的是由三位专家分别标注病灶为”阴性（0）“或”阳性（1）“值；
- prediction-probabilities.csv文件中则是标注每种病灶可能性的概率。

# 四、核心优势特点

1. **临床真实性高**：数据均来源于真实急诊场景，涵盖不同设备、不同扫描参数下的CT影像，数据分布贴合临床落地环境，可有效验证模型的真实泛化能力。

2. **标注权威可靠**：依托三甲资深放射科医师三重交叉校验标注，彻底规避单一医师标注的主观误差，是目前颅脑出血AI模型评测的通用权威基准。

3. **跨中心验证价值突出**：与行业主流训练数据集无数据重叠，是业内公认的外部验证黄金数据集，可精准检验模型的过拟合问题与跨场景泛化能力。

4. **任务适配性广**：全面支持2D切片级分类、3D立体病灶检测、多标签病变识别、颅脑急症筛查等多种主流医学影像研究任务。

# 五、适用研究场景

# 六、数据集局限性（核心注意事项）

以下为CQ500数据集使用的核心注意点，也是科研过程中极易踩坑的关键问题，需重点关注：

- **无像素级分割掩码**：数据集原生仅提供病例级、切片级的分类标签，未配套像素级分割标注，无法直接用于3D病灶分割、精细病灶轮廓分割等分割任务训练。

- **缺乏精细病灶定位标注**：仅支持判断影像内是否存在病变、病变具体类型，无法精准定位病灶位置、轮廓、大小等精细化信息，不适合需要病灶精准定位的高精度研究。

# 七、数据集文件目录与扫描序列说明

## 7\.1 整体目录结构概览

```plain
CQ500_orig/
├── CQ500CT0 CQ500CT0/
│   └── Unknown Study/
│       ├── CT Plain
│       ├── CT PLAIN THIN
│       ├── CT 4cc sec 150cc D3D on
│       ├── CT 4cc sec 150cc D3D on-2
│       └── CT 4cc sec 150cc D3D on-3
├── CQ500CT1 CQ500CT1/
└── ...
```

**CQ500\_orig/**：数据集根目录，用于统一存放所有病例的原始DICOM格式影像文件。

**CQ500CT0 CQ500CT0/**：单病例独立文件夹，每个对应一位患者的一次完整CT检查，CT0、CT1为病例专属编号，用于区分不同样本。

**Unknown Study/**：DICOM医学影像标准层级目录，代表单次完整影像学检查，其内部子文件夹为该次检查下不同参数、不同类型的扫描序列数据。

## 7\.2 各扫描序列文件夹详细含义

### 7\.2\.1 CT Plain（普通平扫CT）

**全称**：Computed Tomography Plain Scan

**序列定义**：无造影剂注入的常规颅脑平扫序列，是CQ500数据集官方标注唯一对应的核心影像序列。

**核心特点**：该序列层厚较大，常规为5\-10mm，是临床颅脑急症筛查最通用的CT扫描方式。数据集所有ICH、IPH、SDH、EDH、SAH等病变标注，均基于该序列切片完成，是训练、验证脑出血分类、检测模型的首选标准数据。

### 7\.2\.2 CT PLAIN THIN（薄层平扫CT）

**全称**：Thin\-slice Plain CT

**序列定义**：同样为无造影剂的纯平扫序列，区别于普通平扫，扫描层厚极薄，常规为1mm及以下。

**核心特点**：空间分辨率远高于普通厚层平扫，可清晰呈现颅内细微结构，能够识别微小骨折、微量出血等易漏诊的细小病灶。多用于影像三维重建、病灶细节分析，也是高精度病灶分割模型的优质训练数据。

### 7\.2\.3 CT 4cc sec 150cc D3D on / \-2 / \-3（增强CT多期扫描序列）

**序列定义**：经静脉注射造影剂后的增强扫描序列，后缀\-2、\-3代表不同的扫描期相，对应动脉期、静脉期、延迟期等不同成像阶段。

**参数释义**：4cc sec为造影剂注射速率（4毫升/秒）；150cc为造影剂总注射剂量（150毫升）；D3D on代表扫描时开启三维重建成像模式。

**核心特点**：通过造影剂强化血管与病灶对比度，可清晰显示颅内血管畸形、动脉瘤、病灶异常强化等病变。需重点注意，CQ500官方的出血、骨折等急症标注均基于平扫序列，此类增强序列**不适用常规脑出血筛查模型训练**，仅可作为颅内血管病变分析的补充研究数据。
