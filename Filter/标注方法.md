# 脑部CT伪影HU阈值与视觉标注规则表
## 一、正常组织HU参考基准
| 组织类型 | HU值区间 | 视觉特征 | 说明                                                        |
| ---- |-------| ---- |-----------------------------------------------------------|
| 脑灰质 | 30~45 | 均匀浅灰色纹理 |                                                           |
| 脑白质 | 22~32 | 灰度略低于灰质，边界清晰 |                                                           |
| 脑脊液 | 0~10  | 深色透亮区域 |                                                           |
| 急性颅内出血 | 50~90 | 高亮致密色块 | 典型急性血肿为50~90 HU，<br/>但超急性期（<2h）可低于50，完全凝固后可达60~85 HU。 |

## 二、各类伪影判定标准
| 伪影类型 | HU判定区间           | 视觉特征 | 标注判定 |
| ---- |------------------| ---- | ---- |
| 金属伪影 | 小于-200 或 或大于1000 | 明暗交错粗大条纹，穿透脑组织，对比度极强 | 判定伪影 |
| 线束硬化伪影 | -40~-10 或 40~80  | 颅骨周边弧形暗环、局部灰度异常偏移 | 判定伪影 |
| 运动伪影 | 运动伪影并不导致真正的CT值系统性偏移  | 图像重影、模糊拖影、细密错乱条纹 | 判定伪影 |
| 部分容积伪影 | 0~50边界异常过渡       | 组织边缘轮廓虚化、灰度渐变失真 | 判定伪影 |
| 环形伪影 | -30~60           | 整张切片出现环状明暗纹路 | 判定伪影 |

## 三、快速筛查阈值规则
1. 绝对异常区间：$\boldsymbol{HU < -20}$ 或 $\boldsymbol{HU > 80}$，极有存在伪影，但需要注意：急性脑出血：可达85~90 HU → 被误判为伪影
2. 临界异常区间：$-20\le HU\le0$ 或 $50\le HU\le80$，结合视觉纹路复核判定
3. 正常区间：$0<HU<50$，无异常纹路即为正常脑组织

## 四、标注实操准则
1. 同时满足数值越界+视觉畸变，标记伪影区域
2. 仅数值小幅波动、无形态纹路异常，不计伪影
3. 出血病灶按病变标注，不归入伪影类别
4. 同一切片多种伪影叠加，统一标记伪影标签
