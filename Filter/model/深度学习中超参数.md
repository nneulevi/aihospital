# 深度学习中的超参数

## 一、核心刚需超参数（直接决定模型精度与收敛效果）

**该板块参数优先级最高，是调参首要优化对象**

### 1\. 学习率 \(Learning Rate, LR\)

- **核心作用**：控制模型权重每次更新的步长，是收敛效果的核心关键

- **参数问题表现**

    - 过大：模型不收敛、损失值震荡、loss爆炸

    - 过小：收敛速度极慢、模型容易陷入局部最优、训练卡顿

- **实战常用区间**：`1e-4 ~ 1e-3`

### 2\. Batch Size（批次大小）

- **核心作用**：每次输入模型的图像数量，影响梯度更新稳定性

- **参数特点**

    - 越大：梯度更新越平稳、模型泛化性更好、精度更高

    - 越小：梯度噪声大、训练震荡明显、稳定性差

- **实战常用值**：`2 / 4 / 8 / 16`（根据显存灵活调整）

### 3\. 训练轮次 \(Epochs\)

- **核心作用**：模型完整遍历整个数据集的次数

- **参数问题表现**

    - 过少：模型训练不充分，出现欠拟合

    - 过多：模型过度学习训练集特征，出现过拟合

- **实战常用区间**：`50 ~ 200`

### 4\. 梯度累计步数 \(Gradient Accumulation Steps\)

- **核心作用**：小显存设备模拟大批次训练效果，解决显存不足问题

- **原理**：多批次累积梯度后再统一更新权重

- **实战常用值**：`2 / 4 / 8`

- **等效批次计算**：等效Batch = 真实BatchSize × 梯度累计步数

## 二、优化器相关超参数（影响收敛速度与过拟合）

### 5\. 优化器类型 \(Optimizer\)

- **Adam**：通用性最强、训练稳定，新手首选

- **AdamW**：优化权重衰减逻辑，防过拟合效果优于Adam，分割任务首选

- **SGD**：调参难度高，收敛慢，但微调后可达到更高精度，竞赛优选

### 6\. 动量 \(Momentum\)

- **适用场景**：仅SGD优化器使用

- **核心作用**：加速模型收敛，弱化梯度震荡

- **实战固定值**：`0.9`

### 7\. 权重衰减 \(Weight Decay\)

- **核心作用**：约束模型权重大小，有效防止过拟合

- **实战常用区间**：`1e-4 ~ 1e-5`

## 三、学习率调度器（LR Scheduler）（控制学习率动态变化）

### 8\. 学习率衰减策略

- **CosineAnnealingLR（余弦退火）**：主流最优策略，学习率平滑波动，适配绝大多数分割任务

- **StepLR**：阶梯式分段下降，参数简单、调参粗放

- **ReduceLROnPlateau**：自适应衰减，验证集指标不提升时自动降低学习率

### 9\. T\_max / step\_size

- **核心作用**：控制学习率下降的速度、周期与幅度

- **适配场景**：T\_max对应余弦退火策略，step\_size对应StepLR策略

## 四、数据增强超参数（低成本大幅提升精度）

**数据增强是图像分割、尤其是医学图像分割提升精度的核心手段，性价比极高**

- 随机旋转

- 随机水平/垂直翻转

- 随机缩放、裁剪

- 高斯噪声、Gamma校正

## 五、模型结构超参数（决定模型容量与显存消耗）

### 10\. 通道数 \(Channels\)

- **UNet标准通道梯度**：64 → 128 → 256 → 512

- **参数特点**：通道数越大，模型特征提取能力越强、模型容量越大，但显存占用越高、训练速度越慢

### 11\. Dropout 率

- **核心作用**：随机失活神经元，强力防止模型过拟合

- **实战常用区间**：`0.1 ~ 0.3`

### 12\. 激活函数

- 常用：ReLU、GELU、LeakyReLU

- 特点：GELU拟合能力更强，ReLU训练更稳定

## 六、损失函数相关超参数（分割任务核心损失配置）

### 13\. 损失函数类型

- **DiceLoss**：图像分割必备，适配正负样本不均衡场景

- **DiceCELoss**：DiceLoss \+ CE损失融合，综合效果最优，实战首选

- **BCEWithLogits**：二分类分割基础损失函数

### 14\. 损失函数权重 \(Lambda\)

- **核心作用**：平衡多种组合损失的权重占比，避免单一损失主导训练

## 七、硬件/训练策略超参数（提速、防过拟合、省显存）

### 15\. 混合精度 \(FP16\)

- **核心作用**：半精度训练，大幅节省显存、加快训练速度，几乎无精度损失

- **实战建议**：所有训练场景默认开启

### 16\. 早停 \(Early Stopping\)

- **核心作用**：监控验证集指标，指标不再提升时自动终止训练

- **价值**：杜绝过拟合、节省训练时间、保留最优模型权重

## 🎯 常用超参数模板

```plaintext
LR = 1e-4
BatchSize = 2
AccumSteps = 4  # 等效Batch=8
Epochs = 100
Optimizer = AdamW
WeightDecay = 1e-5
Scheduler = CosineAnnealing
Loss = DiceCE
```

## 📌 总结

**决定模型训练上限、精度上限的四大核心参数**：

1. 学习率 LR

2. Batch Size 批次大小

3. 梯度累计步数

4. 损失函数配置
